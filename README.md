# Домашнее задание к лекции . «Decorators»

1. logger.py - Обычный декоратор. Пишет в лог аргументы и название функции.
2. logger_param.py - Параметризованный декоратор. Пишет в лог по указанному пути аргументы и название функции.
3. main.py - скрипт для создания книги рецептов и списка покупок. Использует декоратор logger_param для логгирования функции.
